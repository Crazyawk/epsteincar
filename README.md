# Open City Pursuit

A static multi-file browser game built for GitHub Pages.

## Files
- `index.html`
- `style.css`
- `src/` JavaScript modules
- `assets/car-photo.jpg`

## GitHub Pages
1. Upload the whole folder contents to a repo root.
2. In GitHub, open **Settings -> Pages**.
3. Set the source to the main branch root.
4. Wait for the site to publish.

## Notes
- Uses Three.js for rendering.
- Uses cannon-es RaycastVehicle for car physics.
- Works best in a modern browser with WebGL enabled.
